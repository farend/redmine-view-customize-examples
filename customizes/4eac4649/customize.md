# 「個人設定」を非表示にする

ログイン後、画面右上に表示される「個人設定」を非表示します。  

> [!CAUTION]
> 操作している限りメンバーの表示は見えませんが、**HTMLのソースコードには残っています**。

動作確認バージョン：Redmine 6.0 / RedMica 3.2

## 設定

パスのパターン: `.*`

種別: CSS

挿入位置: 全ページのヘッダ

``` css
/* 「個人設定」を非表示にする */
#top-menu a.my-account {
  display: none;
}
```
